-주문 신청이 들어왔지만 상품 출고 준비를 하지 않은 상품들을 말하며, 상품 납품 마감일이 초과될 시 주문이 자동으로 취소가 됩니다.

출고 완료 상품이란?

- 상품 검수가 끝나고 구매자에게 배송 이 되기 바로 직전 및 배송이 완료 된 상품, 구매가 확정된 상품들 입니다.
